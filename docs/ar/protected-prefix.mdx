## البادئة المحمية (عنوان IP العام)

**البادئة المحمية** هي كتلة عناوين IP العامة الخاصة بمؤسستك التي يحميها
Cloud من هجمات DDoS.

| المتطلب | IPv4 | IPv6 |
|-------------|------|------|
| **الحد الأدنى للحجم** | /24 (256 عنوان IP) | /48 |
| **قابلية التوجيه** | قابل للتوجيه العام (ليس RFC 1918) | بث أحادي عالمي (ليس ULA) |
| **التسجيل** | مُعيَّن من سجل الإنترنت الإقليمي (ARIN، RIPE، APNIC) | مُعيَّن من سجل الإنترنت الإقليمي |
| **الإثبات** | خطاب تفويض (LOA) أو تسجيل IRR/ROA | خطاب تفويض (LOA) أو تسجيل IRR/ROA |

**تدفق حركة المرور:**

1. يُعلن Cloud عن بادئتك عالمياً عبر BGP anycast
1. يتم توجيه جميع حركة مرور الإنترنت المتجهة إلى عناوين IP الخاصة بك عبر مراكز التصفية
1. يتم إسقاط حركة مرور الهجمات؛ ويتم تسليم حركة المرور النظيفة عبر أنفاق GRE
